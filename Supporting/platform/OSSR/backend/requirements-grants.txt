# Optional dependencies for the Grant Hunt module.
#
# The grants module is built around a tiered fetcher cascade. You only
# need to install what your target sources actually require:
#
#   ┌──────────────────────────┬─────────────────────────────────────────┐
#   │ backend                  │ when you need it                        │
#   ├──────────────────────────┼─────────────────────────────────────────┤
#   │ scrapling[all] (v2.1+)   │ Primary — tiered HTTP/stealth/dynamic  │
#   │                          │ with TLS-fingerprint impersonation,    │
#   │                          │ Cloudflare Turnstile bypass, adaptive  │
#   │                          │ selectors that survive page redesigns. │
#   │                          │ This is the recommended install.       │
#   ├──────────────────────────┼─────────────────────────────────────────┤
#   │ crawl4ai                 │ Legacy fallback (v0.1 backend). Only    │
#   │                          │ needed if you don't want Scrapling.     │
#   ├──────────────────────────┼─────────────────────────────────────────┤
#   │ httpx + beautifulsoup4   │ Last-resort fallback for static pages   │
#   │                          │ (already shipped in OSSR core deps).    │
#   └──────────────────────────┴─────────────────────────────────────────┘
#
# Install (recommended, full stack):
#     pip install -r backend/requirements-grants.txt
#     scrapling install               # downloads stealth browser profiles
#
# Minimal install (no browser automation, static pages only):
#     pip install httpx beautifulsoup4
#
# Per-source stealth escalation: sources with Cloudflare/SPA walls
# (grants.gov, CORDIS, Horizon Europe, EIC, SBIR) are tagged with
# ``metadata.stealth_level = "stealth"`` or ``"dynamic"`` in sources.py
# and will skip the cheap HTTP path, routing directly to the Scrapling
# browser rung.
